# Title
Robert Dictionary

# Description.
Robert Dictionary is  an interactive dictionary web application built using HTML, CSS and Javascript. it retrives word definitions,phonetic pronouciations,audio pronouciation,synonyms and source references from public dictionary API. users can search for words dynamically and store their favorite words using local storage for easy access later.

# Tools
This project has been built using:
1. HTML
2. CSS
3. Javascript
4. Dictionary API

# Setup
follow these steps to set up and run Robert Dictionary
1. clone the Repository; clone the Robert Dictionary repository to your local machine using git.

git clone https://github.com/robert00-1/robert_dictionary.git

2. Navigate into the project folder
  cd robert_Dictionary

 3. enter  the command ls to see the list of items inside my folder
 4. enter the command code . to open my file in vs code
 5. right click the index.html to open liveserve and you will see Robert Dictionary 
 6. start using the app
  . Enter a word in the search box.
  .Click the "Search"button to see definitions,synonyms and audio.
  .Click"Save Favorite" to add a word to your favorites list.
